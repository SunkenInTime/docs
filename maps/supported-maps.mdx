## Map Assets

Each map includes:

- High-resolution SVG base maps
- Attack and Defense perspectives  
- Callout overlays with location names
- Spawn barrier positions
- Ultimate orb locations
- Thumbnail preview images

## Creating Strategies

To use a map for strategy creation:

1. Open the map selector in the top navigation
2. Choose your desired map from the list
3. Select Attack or Defense side using the sword/shield toggle
4. Begin placing agents, utilities, and annotations

All maps support the full range of Icarus features including agent placement, ability lineups, custom drawings, text annotations, and image uploads.
